# credit-card-fraud-detection
End-to-end Credit Card Fraud Detection using MySQL, Python EDA, and Power BI Dashboard
## 📌 Project Description

This project analyzes 100,000 credit card transactions 
to detect fraudulent activity using a complete 
data analytics pipeline.

Starting from raw data in MySQL, performing deep 
Exploratory Data Analysis in Python, and finally 
visualizing insights in an interactive Power BI dashboard.

## 🎯 Objective
Identify patterns, trends, and geographic distribution 
of fraud transactions to help detect financial crime.

## 📊 Dataset
- Total Transactions : 100,000
- Fraud Cases        : 1,000
- Fraud Rate         : 1%
- Total Amount       : $249.71M
- Avg Fraud Amount   : $2.52K

## 🛠️ Tools & Tech
- MySQL Workbench  → Data storage & SQL queries
- Python           → EDA (Pandas, Seaborn, Matplotlib)
- Power BI         → Dashboard & Visualization

## 💡 Key Insights
- Fraud spikes vary month to month
- Geographic fraud clusters across the US
- Avg fraud & legit amounts are surprisingly similar
